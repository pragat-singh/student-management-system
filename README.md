# 📚 Student Management System
A web-based Student Management System developed using **Node.js, Express.js, MySQL, EJS, CSS, and JavaScript** to manage student records, attendance, and marks efficiently through an admin dashboard.

--

## 🚀 Features
- 🔐 Admin Login Authentication
- 📊 Dashboard with summary statistics
- ➕ Add Student Records
- 📋 View Student List
- ✏️ Edit Student Details
- ❌ Delete Student Records
- 🗓️ Add Attendance Records
- 👀 View Attendance Records
- 📝 Add Student Marks
- 📈 View Marks Records
- 🔄 Update Attendance & Marks
- 🚪 Logout Functionality

## ⚙️ Installation Steps

Follow these steps to run the project locally:

### 1️⃣ Clone the repository
git clone https://github.com/pragat-singh/student-management-system.git
### 2️⃣ Navigate to project folder
cd student-management-system

### 3️⃣ Install dependencies
npm install
### 4️⃣ Setup MySQL Database
Create database:

CREATE DATABASE student_management;
Import required tables (students, attendance, marks, admin).

### 5️⃣ Run the application
node app.js

Server will start at:
http://localhost:3000
